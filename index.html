About the author

In March 2020, Mark having been unsuccessful in recent job searches and applications undertook a Code Nation code developers training course in Manchester. Although Mark already has experience in other programming languages (primarily Visual Basic/VB.Net) he could see the benefit in re-skilling to fit in with the latest demand in the marketplace for web and mobile application developers. The course undertaken by Mark was Develop:Coding which is designed to take the student through the fundamentals of programming in HTML, CSS, JavaScript, get to grips with front-end and mobile application development, and find out what it’s like to work as a full-fledged professional coder.
